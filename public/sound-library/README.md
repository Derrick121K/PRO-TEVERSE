# PRO-TEVERSE Sound Library

This folder contains legal starter samples generated by the project script:

tools/generate-starter-samples.mjs

These starter sounds are synthetic and created for this repository.

You may replace or add sounds only if you have legal permission to use and redistribute them.

Allowed:
- Your own recordings
- Your own exported stems
- CC0 sounds
- Public domain sounds
- Royalty-free sounds that allow redistribution

Not allowed:
- Paid FL Studio stock sounds copied into this repo
- Cracked plugin sounds
- Pirated sample packs
- Copyrighted songs without permission
